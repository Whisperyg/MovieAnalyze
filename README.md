# MovieAnalyze
使用Spark对《邪不压正》电影的爬取豆瓣评分进行分词，用户的情感分析，使用d3-Cloud将结果数据以词云的方式展现出来，最后通过分词用户对该电影的评分来判断该电影的口碑。
